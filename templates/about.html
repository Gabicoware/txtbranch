About

txtbranch is a collaborative writing environment. Like a wiki, it is meant to allow any user to contribute. A set of system rules is meant to organize the collaboration.

Alpha

We're in alpha. Access to this website may be limited or withdrawn at any time.

Definitions

Tree - a tree is a story, or rather set of stories that originate from a single point. A tree is made of many branches, which split or continue the previous branch. Every tree has a first or root branch.

Branch - a branch is a segment on a tree. On txtbranch, a branch is composed of a teaser which readers can see when they are selecting between branches, and the text which contains the rest of the branch.

How to get started

Click on home and open one of the trees. Select the branch that you think is best. follow that until there are no more branches to open. You can then go back earlier in the tree and follow a different branch.
